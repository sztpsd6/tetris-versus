# Tetris Versus

基于 HTML5 Canvas 和 Socket.io 开发的在线双人俄罗斯方块对战小游戏。

## 游戏特点
- 实时双人在线对战
- 基础操作：旋转、左右移动、加速下落、直接到底
- 攻击机制：一次性消除多行（2行及以上），可给对手增加对应的垃圾行

## 运行方法
1. 安装依赖：`npm install`
2. 启动服务：`npm start`
3. 访问网页：`http://localhost:3000`

## 怎么玩？
1. 两人分别打开网页。
2. 输入相同的房间号，点击“加入房间”。
3. 两人都加入后，游戏自动开始！
